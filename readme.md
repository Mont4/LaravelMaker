## Installation

Via Composer

``` bash
$ composer require mont4/laravelmaker
$ php artisan vendor:publish --provider="Mont4\LaravelMaker\LaravelMakerServiceProvider" --tag="migrations"
```

## Usage

``` bash
$ php artisan make:all Example/Example
$ php artisan permission:sync
```
